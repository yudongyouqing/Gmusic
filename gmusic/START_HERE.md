# 从这里开始！

欢迎来到 **GMusic** - 一个专为大厂实习设计的 Golang 本地音乐播放器项目！

## ⚡ 30 秒快速了解

这是一个**完整的全栈项目**，包含：

- 🎧 **Golang 后端**：音频播放、元数据提取、媒体库索引
- 🎨 **React 前端**：现代化 UI、实时歌词、播放控制
- 💾 **SQLite 数据库**：媒体管理、搜索、播放历史
- [object Object] WebSocket**：完整的后端接口

**代码量**：~2000 行（包括注释）  
**技术栈**：Golang + React + SQLite + Docker  
**难度**：⭐⭐⭐⭐ 中高级

## 🚀 5 分钟快速启动

### 前置要求
- Go 1.21+
- Node.js 16+
- 音乐文件（MP3、FLAC 等）

### 启动步骤

**步骤 1**：安装依赖
```bash
cd gmusic
make install-deps
```

**步骤 2**：启动后端（终端 1）
```bash
make dev
```

**步骤 3**：启动前端（终端 2）
```bash
make frontend
```

**步骤 4**：打开浏览器
访问 `http://localhost:5173`

✅ 完成！
