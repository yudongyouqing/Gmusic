## 🎯 项目核心功能

### 音频播放
✅ 支持格式：MP3、FLAC、WAV、AAC  
✅ 播放控制：播放、暂停、停止、恢复  
✅ 音量控制：0-100%  
✅ 进度显示：实时位置和总时长  

### 元数据提取
✅ 歌曲信息：标题、歌手、专辑  
✅ 封面提取：从 ID3v2/FLAC metadata 中提取  
✅ 时长计算：精确到秒  
✅ 比特率识别：自动检测  

### 歌词功能
✅ LRC 格式解析  
✅ 时间轴同步显示  
✅ 歌词滚动窗口  
✅ 当前行高亮  

### 媒体库系统
✅ 目录扫描：递归遍历文件  
✅ 并发索引：工作池模式  
✅ 数据库存储：SQLite + GORM  
✅ 搜索功能：按名字、歌手、专辑  

### 前端 UI
✅ 现代化界面：渐变色、圆角、阴影  
✅ 响应式设计：桌面和平板适配  
✅ 实时更新：WebSocket 同步  
✅ 搜索功能：实时搜索结果

- SQLite + GORM（数据库）
- Gin（Web 框架）

### 前端（React）
- React 18
- Axios（HTTP）
